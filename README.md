# ☁️ МетеоБот / Weather Telegram Bot

## 📖 Описание проекта
**МетеоБот** — это удобный и интуитивно понятный Telegram-бот для отслеживания прогноза погоды. Бот полностью управляется кнопочным интерфейсом.

## ✨ Основные возможности
* 🌡️ **Текущая погода** — моментальное отображение температуры и реальных ощущений.
* ⚙️ **Гибкие настройки** — быстрая смена города "в один клик".
* 🤖 **Умный UI** — динамические клавиатуры, которые меняются в зависимости от контекста диалога.

## 🛠️ Технологический стек
* 🐍 **Язык программирования:** Python (v3.12.10)
* ⚡ **Веб-Фреймворк:** FastAPI (v0.137.1)
* 🤖 **Библиотека бота:** python-telegram-bot (v22.8)
* 🗄️ **База данных:** Redis (v8.8.0) 
* 🌤️ **Погодный API:** OpenWeatherMap / или ваш вариант
* 🐳 **Контейнеризация:** Docker (v4.78.0)

## 🏗️ Архитектура проекта

* `telegram_bot` — бот-обработчик команд пользователя (управляет диалогом через кнопки). Берет данные через сервер `weather_api`.
* `weather_api` — эндпоинты, бизнес-логика, запросы к Redis-хранилищу.
* Фоновый воркер — обращается к API и обновляет данные в Redis-хранилище раз в определенное время. 
* Redis-лимитер (slowapi) — ограничение запросов к серверу `weather_api`.
* Redis-хранилище — хранилище текущих данных о погоде, получает данные от фонового воркера.

## 🚀 Быстрый запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/danilpashin/weather-chatbot.git
cd weather-chatbot
```

### 2. Настройка окружения
Создайте `.env` файл в корне проекта, скопировав `.env.example` файл. Вставьте свои токены API и Telegram-бота:
```bash
cp .env.example .env
``` 

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск Redis-контейнеров (лимитер и основное хранилище)
```bash
docker run --name redis-limiter -d -p 6379:6379 redis
docker run --name redis-main-storage -d -p 6380:6379 redis
docker run --name redis-backup-storage -d -p 6381:6379 redis
```

### 5. Запуск сервера с данными о погоде (uvicorn)
В корневой папке проекта введите команду:
```bash
python -m weather_api.main --reload
```

### 6. Запуск бота
В корневой папке проекта введите команду:
```bash
python -m telegram_bot.handlers.main
```

### 7. Использование бота
Бот запускается через команду `/start`
